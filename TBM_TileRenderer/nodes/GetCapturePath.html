Get Capture Path

Gets the path on the disk to save the captured images at

Inputs

In
Exec

Target
TBM Tile Renderer Object Reference

Outputs

Out
Exec

Out Path
String

Return Value
Boolean

false if failed to find a valid path The path for captured images is <ProjectSavedDir/TBM/Level/Color/> where LevelTag is the last part of LevelTag and Color is the last part of ColorTag (or 'Default' if not specified) LevelTag is required to be valid

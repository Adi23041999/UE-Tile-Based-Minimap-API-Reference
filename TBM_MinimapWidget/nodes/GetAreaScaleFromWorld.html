Get Area Scale from World

Gets an area blip scale from world space dimensions. Eg. from a box or sphere radius
The result should be divided by the blip image texture size

Inputs

Target
TBM Minimap Widget Object Reference

World Size
Vector 2D Structure

World dimensions of the area (box or sphere)

Outputs

Return Value
Vector 2D Structure

UI blip scale

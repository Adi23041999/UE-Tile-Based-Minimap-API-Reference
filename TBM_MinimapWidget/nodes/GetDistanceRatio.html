Get Distance Ratio

Gets the distance from the center to the specified location as 0-1 range (0 being the minimap center and 1 being the border of the minimap)

Inputs

Target
TBM Minimap Widget Object Reference

Position
Vector 2D Structure

the UI position to calculate the value for

Outputs

Return Value
Float (double-precision)

minimap center to closest border value as 0-1 range

Event FindBlipImageWidget

Finds or creates an image widget for the blip. Needs to be overridden if the user does not intend to use an image for the blip
NOTES: by default the image can be found either as child of this widget or a child of another child widget
The widgets needs to have a specific name in order to be found. See FTBMConstants struct to check the names
